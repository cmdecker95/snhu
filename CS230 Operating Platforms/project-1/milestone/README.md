# CS 230 Project One Milestone - Singleton Pattern

Milestone version focused on singleton pattern for GameService. UML diagram of singleton structure.

## Lessons Learned

- **[Java, Singleton]:** Static field, private ctor, public `getInstance`. Simple but easy to break (thread-safety, reflection).
- **[Java, UML]:** Diagrammed singleton with `-instance: GameService` and `+getInstance(): GameService`. Visual spec aids team communication.
- **[Java, Eclipse]:** FIXME tags in starter code direct attention. Productivity tool, not just comments.
