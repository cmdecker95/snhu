# CS 230 Module Four - GameAuth Login UI

React + TypeScript + Vite frontend for game user authentication. Login form with typed API integration.

## Lessons Learned

- **[React, TypeScript]:** TypeScript catches API response shape mismatches at compile time. Fewer runtime surprises.
- **[Vite]:** Fast dev server with HMR. No bundler config needed for standard setups.
- **[React, SPA Auth]:** Login form captures credentials, sends to Dropwizard backend. Frontend-backend separation clarifies data flow.
- **[CSS, UI Polish]:** Styled login screen matches Gaming Room brand. Presentation matters even in prototypes.
