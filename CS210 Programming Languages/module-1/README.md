# CS 210 Module One - Hello World (C++ & Python)

First C++ and Python programs. Set up Visual Studio (C++) and PyCharm (Python). Compared language ergonomics.

## Lessons Learned

- **[C++, Python]:** C++ needs boilerplate (`#include <iostream>`, `int main()`). Python is one `print()`. Each has place.
- **[VS, PyCharm]:** IDE setup matters. VS heavyweight but debugs well. PyCharm lighter for Python.
- **[Multi-language]:** Knowing 2+ languages reveals what's language-specific vs general CS.
